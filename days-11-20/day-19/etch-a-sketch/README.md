# Day 19: Etch-a-Sketch

A simplified etch-a-sketch screen where you can draw with one control. 

Controls:
- [ `W` ]: move forward
- [ `S` ]: move backward
- [ `A` ]: rotate left
- [ `D` ]: rotate right
